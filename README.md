# 📝 CompactCSS Tool

**CompactCSS** is a fast, lightweight, and user-friendly web application designed to help developers compress, format, and optimize their CSS code. Built with **React** and **Vite**, it offers real-time conversions and multiple formatting options to suit your specific project needs.

![CompactCSS Preview](https://via.placeholder.com/800x400?text=CompactCSS+Preview+Image) <!-- เปลี่ยน URL รูปภาพนี้เป็นรูป Screenshot จริงของเว็บคุณ -->

## ✨ Features

- **Multiple Formatting Options:**
  - **Compact:** Formats CSS into a readable single-line-per-rule style (Scannable).
  - **Minify:** Aggressively removes all whitespace for production-ready, ultra-small file sizes.
  - **Pretty:** Beautifully indents and formats messy CSS into a standard, readable structure.
- **Comment Preservation:** Choose whether to keep or remove `/* comments */` during conversion.
- **Drag & Drop Support:** Easily upload your `.css` files by dragging them directly into the browser.
- **Smart Analytics:** View real-time statistics on input size, output size, and total size reduction percentage.
- **Quick Export:** Copy the converted CSS to your clipboard with one click, or download it directly as a `.css` file.
- **Dual Language & Theme:** Supports English (EN) and Thai (TH), along with a seamless Light/Dark mode toggle that remembers your preferences.

## 🚀 Tech Stack

- **Framework:** [React 19](https://react.dev/) (Using React Compiler)
- **Build Tool:** [Vite](https://vitejs.dev/) (For lightning-fast HMR and building)
- **Routing:** [React Router v6](https://reactrouter.com/)
- **Icons:** [Lucide React / React Icons (Fi)](https://lucide.dev/)
- **Styling:** Custom Vanilla CSS (Written in Compact CSS methodology)
